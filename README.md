# Leon Koome — Portfolio

Personal portfolio site for Leon Koome — software engineer. Built with Next.js (app router), TypeScript, Tailwind CSS, and Framer Motion.

```bash
npm install
npm run dev
```

## Structure

- `app/` — root layout, page, metadata, favicon
- `src/components/` — Navbar, Hero, Marquee, About, Skills, Projects, CTA, Contact, Footer, CommandPalette, Cursor, CanvasBackground
- `pages/api/contact.ts` — contact form endpoint
- `public/` — project preview images, résumé, OG share image
