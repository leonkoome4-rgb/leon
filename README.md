# Leon — Futuristic Portfolio (Scaffold)

This repository is a scaffold for a high-end personal portfolio using Next.js, TypeScript, Tailwind CSS, Framer Motion, and React Three Fiber.

Quick start:

```bash
# install
npm install

# dev
npm run dev
```

What I scaffolded:
- Next.js app router layout and homepage
- `Hero`, `CanvasBackground`, `Skills`, and `Projects` components
- Tailwind CSS + PostCSS
- Simple contact API route
